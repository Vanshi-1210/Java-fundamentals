# ☕ Java Fundamentals

This repository contains Java fundamental concepts and practice programs designed to build a strong foundation in Java programming.

It includes beginner-friendly examples covering core concepts, logic building, and basic problem-solving.

---

## 📌 Topics Covered

### Basics
- Introduction to Java
- Hello World Program
- Variables and Data Types
- Input / Output
- Operators

### Control Flow
- Conditional Statements
- Switch Case
- Loops

### Pattern Practice
- Solid Rectangle
- Hollow Rectangle
- Half Pyramid
- Number Patterns

### Functions
- Function Basics
- Parameter Passing
- Return Types
- Practice Problems

---

## 🎯 Purpose

This repository is created to:

- Build a strong Java foundation
- Practice core programming concepts
- Improve logical thinking
- Prepare for Data Structures and Algorithms

---

## 💻 Structure

Each concept is implemented using clean and beginner-friendly Java programs.

---

## 🚀 How to Run

1. Clone the repository
2. Open in any Java IDE (VS Code / IntelliJ / Eclipse)
3. Compile and run the `.java` files
